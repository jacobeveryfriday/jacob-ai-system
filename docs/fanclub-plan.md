## 2. 세계관 (Universe)

### 컨셉: "500달톤 유니버스"

밀리밀리의 핵심 기술인 **500달톤 저분자 기술**을 세계관의 중심으로 설정합니다.

> "피부에는 보이지 않는 문이 있습니다.
> 500달톤보다 큰 것은 통과할 수 없는 문.
> 밀리밀리는 그 문을 열 수 있는 열쇠를 만듭니다."

### 세계관 키워드
- **500달톤 게이트**: 피부 흡수의 관문 (과학적 사실 기반)
- **밀리 가디언**: 밀리 스킨 멤버들의 별칭
- **스킨 저널**: 멤버들이 기록하는 피부 변화 일지
- **달톤 랩**: 밀리밀리 R&D 비하인드를 공유하는 공간

### 멤버 등급 체계

| 등급 | 조건 | 아이콘 |
|------|------|--------|
| **Seed (씨앗)** | 가입 즉시 | 🌱 |
| **Sprout (새싹)** | 첫 구매 완료 | 🌿 |
| **Bloom (블룸)** | 3회 이상 구매 | 🌸 |
| **Garden (가든)** | 6회 이상 구매 + 리뷰 3건 | 🌺 |
| **Guardian (가디언)** | 12회 이상 + 콘텐츠 제작 | 🏅 |

## 3. 멤버 혜택

### 전체 멤버 공통 혜택
- 신제품 출시 1주일 전 사전 공개
- 월 1회 "밀리 레터" (스킨케어 팁 + 비하인드)
- 밀리 스킨 전용 이벤트 참여 자격

### 등급별 추가 혜택

| 등급 | 혜택 |
|------|------|
| Sprout | 다음 구매 10% 할인 쿠폰 |
| Bloom | 매월 미니 샘플 키트 증정 + 15% 할인 |
| Garden | 신제품 체험단 우선 선발 + 20% 할인 |
| Guardian | R&D 미팅 참여 + 연 1회 풀사이즈 키트 증정 + VIP 25% 할인 |

### 특별 혜택
- **생일 혜택**: 생일 월 30% 할인 + 한정 파우치 증정
- **친구 초대**: 1명 초대 시 양쪽 모두 5,000원 쿠폰
- **스킨 저널 보상**: 피부 변화 후기 업로드 시 포인트 적립

## 4. TikTok 챌린지

### 챌린지명: #500달톤챌린지 (#500DaltonChallenge)

**컨셉**: "500달톤이 피부에 들어가는 순간"을 표현하는 Before/After 챌린지

**참여 방법**:
1. 밀리밀리 제품 사용 전 셀카 촬영
2. 제품 사용하는 모습 촬영 (트랜지션)
3. 사용 후 "글로우업" 모습 촬영
4. 해시태그 #500달톤챌린지 #밀리밀리 #MiliSkin 태그
5. 밀리밀리 공식 계정 태그

**챌린지 BGM**: 오리지널 사운드 제작 (15초, 몽환적 + 청량한 분위기)

**보상 체계**:

| 순위 | 보상 |
|------|------|
| 1위 (조회수 기준) | 밀리밀리 전 제품 풀세트 + 50만원 상당 |
| 2~5위 | 500달톤 시리즈 세트 + 15만원 상당 |
| 6~20위 | 베스트 제품 1개 + 5만원 상당 |
| 참여자 전원 | 밀리밀리 20% 할인 쿠폰 |

**목표 수치**:
- 참여 영상: 500건 이상
- 총 조회수: 1,000만 회
- 해시태그 노출: 5,000만 회
- 신규 팔로워 유입: 3,000명

## 6. 대사(앰배서더) 조건

### 밀리 앰배서더 — "밀리 가디언 앰배서더"

**선발 조건:**

| 항목 | 기준 |
|------|------|
| 팬클럽 등급 | Garden 이상 |
| 구매 이력 | 6회 이상 구매 |
| 콘텐츠 활동 | 리뷰 3건 이상 OR 숏폼 2건 이상 |
| SNS 팔로워 | 1,000명 이상 (필수 아님, 가산점) |
| 활동 기간 | 팬클럽 가입 후 2개월 이상 |

**앰배서더 혜택:**

| 혜택 | 내용 |
|------|------|
| 제품 지원 | 매월 신제품 풀사이즈 무료 제공 |
| 할인 | 상시 30% 할인 |
| 수익 공유 | 개인 할인 코드 발급, 판매당 10% 커미션 |
| 참여 기회 | R&D 미팅, 신제품 네이밍 투표, 촬영 참여 |
| 인증 | 공식 앰배서더 배지 + 프로필 소개 |

**활동 의무:**

| 활동 | 주기 | 내용 |
|------|------|------|
| 인스타그램 포스트 | 월 2회 | 제품 사용 후기 |
| TikTok/Reels | 월 1회 | 숏폼 콘텐츠 |
| 리뷰 작성 | 신제품 출시 시 | 구매 플랫폼 리뷰 |
| 커뮤니티 참여 | 상시 | 카카오 오픈채팅 활동 |

**선발 프로세스:**
1. 신청서 접수 (구글폼)
2. 구매/활동 이력 검증
3. 1차 선발 (10명)
4. 1개월 수습 기간
5. 최종 앰배서더 확정 (5~7명)
