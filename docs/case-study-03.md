# K-뷰티 케이스스터디 #3
## 어필리에이트 마케팅으로 연 8억 달성 — C 브랜드

---

### 문제

C 브랜드(선크림/베이스 메이크업)는 자사몰과 네이버 스마트스토어 중심으로 운영 중이었습니다.

- 연 매출: **3.2억원** (자사몰 60%, 네이버 40%)
- 마케팅 비용: 월 **800만원** (검색광고 + SNS 광고)
- 광고 ROAS: **2.8x** (수익성 한계)
- 어필리에이트/제휴 채널: **0개**
- 해외 매출 비중: **0%**

핵심 원인: 자사 채널 의존도 과다, 퍼포먼스 광고 한계, 제휴 채널 미개척

---

### 솔루션 — 08Liter Enterprise 패키지 + 어필리에이트 셋업

**Phase 1 (1~3개월): 어필리에이트 인프라 구축**
- 국내 어필리에이트 네트워크 셋업 (링크프라이스, 애드픽)
- 해외 어필리에이트 셋업 (아마존 어소시에이츠, 쇼피 어필리에이트)
- 수수료 구조 설계: 국내 8%, 해외 12%
- 어필리에이트 전용 랜딩페이지 + 배너 제작

**Phase 2 (4~6개월): 인플루언서 어필리에이트 확대**
- 뷰티 블로거/유튜버 50명에게 어필리에이트 링크 배포
- 성과 기반 인센티브 (월 100만원 이상 발생 시 수수료 +3%)
- 숏폼 100건 + 구매평 100건으로 전환율 뒷받침

**Phase 3 (7~12개월): 글로벌 스케일업**
- 동남아 어필리에이터 200명 네트워크 확장
- 아마존 US/JP 입점 + 어필리에이트 연동
- 자동화 대시보드로 실시간 성과 트래킹
