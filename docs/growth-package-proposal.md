# 08Liter Global — Growth 패키지 제안서

> **"브랜드 성장의 모든 것을 하나의 패키지로"**
> 구매평 + 숏폼 + 맞춤 인플루언서 — 검증된 3-in-1 솔루션

---

## 1. Growth 패키지 개요

Growth 패키지는 단순 리뷰 대행을 넘어, **숏폼 콘텐츠 + 인플루언서 매칭 + 성과 분석**을 결합한 통합 마케팅 솔루션입니다.

기존 Starter 패키지(구매평 단건/숏폼 20건)에서 한 단계 진화하여, 브랜드의 실질적 매출 증가를 목표로 설계되었습니다.

---

## 2. 패키지 3티어 상세

### 2-1. Basic Growth — 450만원/월

| 항목 | 내용 |
|------|------|
| 구매평 작성 | 30건 (네이버/쿠팡/올리브영) |
| 숏폼 콘텐츠 | 50건 (릴스/쇼츠/틱톡) |
| 인플루언서 매칭 | 마이크로 1명 (팔로워 1만~5만) |
| 성과 리포트 | 월 1회 기본 리포트 |
| 캠페인 기간 | 30일 |
| 수정 횟수 | 콘텐츠당 1회 |
| 담당 매니저 | 공용 매니저 배정 |

**적합 대상:** 온라인 입점 초기, 리뷰 기반 확보가 급한 브랜드

### 2-2. Standard Growth — 520만원/월 ⭐ 추천

| 항목 | 내용 |
|------|------|
| 구매평 작성 | 50건 (네이버/쿠팡/올리브영/아마존) |
| 숏폼 콘텐츠 | 50건 + 롱폼 리뷰 5건 |
| 인플루언서 매칭 | 맞춤 인플루언서 1명 (팔로워 5만~20만) |
| 성과 리포트 | 격주 리포트 + 중간 전략 미팅 1회 |
| 캠페인 기간 | 45일 |
| 수정 횟수 | 콘텐츠당 2회 |
| 담당 매니저 | 전담 매니저 1:1 배정 |
| 추가 혜택 | 경쟁사 벤치마크 분석 1회 |

**적합 대상:** 매출 성장 가속이 필요한 중소 브랜드, 해외 진출 준비 브랜드

### 2-3. Premium Growth — 600만원/월

| 항목 | 내용 |
|------|------|
| 구매평 작성 | 80건 (국내 + 글로벌 4개 플랫폼) |
| 숏폼 콘텐츠 | 70건 + 롱폼 리뷰 10건 |
| 인플루언서 매칭 | 맞춤 인플루언서 2명 (매크로 1명 포함) |
| 성과 리포트 | 주간 리포트 + 전략 미팅 월 2회 |
| 캠페인 기간 | 60일 |
| 수정 횟수 | 무제한 |
| 담당 매니저 | 시니어 매니저 + 어시스턴트 |
| 추가 혜택 | A/B 테스트 소재 제작, 어필리에이트 셋업 지원 |

**적합 대상:** 카테고리 1위 목표, 글로벌 동시 런칭 브랜드

## 3. ROI 시뮬레이션

### Standard Growth 520만원 투자 시 기대 수익

```
투자금: 520만원

[직접 효과]
  구매평 50건 → 전환율 1.8%p 상승 (업계 평균 기준)
  숏폼 50건 → 월 노출 250만 회 (평균 도달 5만/건)
  인플루언서 1명 → 추가 노출 30만 회

[매출 환산]
  기존 월매출: 3,000만원 (가정)
  전환율 상승 효과: +540만원 (3,000만 × 1.8%)
  숏폼 유입 효과: +380만원 (250만 노출 × 0.3% CTR × 5만원 AOV)
  인플루언서 효과: +220만원 (30만 노출 × 0.4% CTR × 18만원 AOV)

  기대 매출 증가: +1,140만원/월
  ROI: 520만원 투자 → 1,140만원 증가 = 219% ROAS
```

### 3개월 누적 시뮬레이션

| 월 | 투자 | 기대 매출 증가 | 누적 ROI |
|----|------|---------------|----------|
| 1개월 | 520만원 | +1,140만원 | 219% |
| 2개월 | 1,040만원 | +2,510만원 (리뷰 누적 효과) | 241% |
| 3개월 | 1,560만원 | +4,200만원 (브랜드 인지도 효과) | 269% |

> **3개월 기준 투자 대비 2.7배 매출 증가 기대**

## 4. 경쟁사 대비 차별점

### 차별점 1: 데이터 기반 인플루언서 매칭
- 단순 팔로워 수가 아닌, **실제 구매 전환율** 기준 매칭
- 08Liter 자체 DB 12,000+ 인플루언서 성과 데이터 축적
- 경쟁사: 팔로워 수 기준 수동 매칭 → 전환율 편차 큼

### 차별점 2: 숏폼 + 구매평 동시 진행
- 숏폼으로 인지도 → 구매평으로 전환 → 동시 시너지
- 경쟁사: 구매평 대행 OR 숏폼 대행 → 별도 계약, 별도 비용
- 08Liter: **하나의 패키지에서 풀퍼널 커버**

### 차별점 3: 성과 보장 + 추가 비용 없는 수정
- Standard 이상: 콘텐츠 수정 2회 이상 무료
- 목표 도달률 미달 시 추가 콘텐츠 보상 제공
- 경쟁사: 수정 시 추가 비용 발생, 성과 미보장
